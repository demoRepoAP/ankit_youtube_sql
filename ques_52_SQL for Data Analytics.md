Solution:
Part 1:
```sql
select u.segment, count(distinct u.user_id) as total_user_count,
count(distinct case when line_of_business = 'Flight' and month(b.booking_date) = 4 then b.user_id end) as user_who_booked_april
from booking_table as b
right join user_table as u on b.user_id = u.user_id
group by u.segment
```

Part 2:

```sql
select * from booking_table

select user_id from (select user_id, min(booking_date) as first_date, line_of_business from booking_table
group by user_id)
as a  where a.line_of_business = 'Hotel'
```

part 3:

```sql
with cte as (
select user_id, min(booking_date) as mini, max(booking_date) as maxi  from booking_table
group by user_id
)

select user_id, datediff(maxi, mini) as day_bet_first_last_booking from cte
```

part 4:
```sql

select  segment,
sum(case when line_of_business = 'flight' then 1 else 0 end) as flight_buss,
sum(case when line_of_business = 'Hotel' then 1 else 0 end) as flight_buss
from booking_table as b
inner join user_table as u on b.user_id = u.user_id
group by segment
```
